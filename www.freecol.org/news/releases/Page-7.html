FreeCol 0.7.2 Released
Wednesday, 19 September 2007
We have now released version 0.7.2 of FreeCol. This release fixes several bugs and adds some new features as well. Some of the major new features are:
  • The game can now switch between fullscreen and windowed mode.
  • The application window can be resized.
  • Zooming on the mapboard has been implemented.
 
FreeCol 0.7.1 Released
Monday, 13 August 2007
This release fixes the bug causing native units to be frozen on the mapboard. Read the complete list of bugfixes and new features.
 
FreeCol 0.7.0 Released
Friday, 13 July 2007
  • Trade and negotiations between human players.
  • A simple map editor has been created.
  • The in-game user documentation has been improved.
  • Map of the Americas added.
  • Automated trade routes.
  • Lots of bugfixes and more features.
 
FreeCol 0.6.1 Released
Monday, 30 April 2007
  • Clicking a colony name on the Turn Report will now display that Colony.
  • A panel for displaying the requirements of the colonies has now been added.
  • The bug causing the game to deadlock after displaying the emigration panel has been fixed.
  • Multiplayer games should now be stable after fixing two critical bugs.
  • Some other bug fixes as well.
 
FreeCol 0.6.0 Released
Monday, 16 April 2007
  • The user interface has been improved.
  • Reports have been added.
  • Dialogs can now be resized and moved.
  • Tons of other new features and bug fixes.
